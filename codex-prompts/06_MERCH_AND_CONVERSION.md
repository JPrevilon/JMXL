# Prompt 6 — Merch showroom and conversion system

Complete the merch section using approved products, mockups, prices and store links.

Requirements:
- lead with three offers: statement tee, premium item and album bundle
- show product name, price, short value statement, available sizes/status and CTA
- build a polished hover/tilt treatment that is disabled on touch and reduced-motion devices
- make the bundle visually strongest without turning the page into a generic ecommerce grid
- link to the existing merch store unless a local cart integration is explicitly approved
- track outbound clicks through a lightweight analytics helper
- add sticky mobile CTAs for “Buy Album” and “Shop Merch” after the visitor passes the hero
- avoid fake inventory scarcity; only display limited/sold-out labels from actual data
- ensure CTA contrast and keyboard usability

Keep product content in `src/data/products.ts`. Run typecheck, lint and build.
