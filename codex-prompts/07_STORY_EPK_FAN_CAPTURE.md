# Prompt 7 — Story, EPK, booking and fan capture

Complete the artist story, EPK drawer and fan capture sections.

Story:
- integrate approved short bio, long bio, Los Angeles identity and project story
- use 2–3 portraits that transition as the story advances
- include only verified career highlights and press quotes

EPK:
- add approved one-sheet PDF, press photo download, music links, credits and booking contact
- drawer must be keyboard accessible with focus trapping and return focus on close
- allow direct linking to `#epk`

Fan capture:
- connect the form to the selected email/SMS provider through a secure route or provider embed
- include success, error, loading and consent states
- never expose private API keys client-side
- add spam protection appropriate to the selected provider
- preserve a simple three-field maximum initial form

Run typecheck, lint and build.
